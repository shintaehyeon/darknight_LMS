# 🦇 Dark Knight SaaS 2.0 - 인수인계서 (Handover Document)

> [!NOTE]
> 본 프로젝트는 기존 크롬 확장 프로그램에서 **AI RAG 에듀테크 SaaS**로 진화하는 2.0 버전을 구축 중입니다.
> 현재 이 파일이 있는 `dark-knight` 저장소의 `2.0` 브랜치가 **모노레포(Monorepo)** 역할을 합니다.

## 📁 디렉토리 구조 (Monorepo)
- `/` (루트) : 1.0 버전이었던 크롬 익스텐션 코드 (SaaS 연동을 위해 `popup.js`가 수정됨)
- `/darkknight-saas` : 프론트엔드 대시보드 (Next.js 15+, App Router, next-pwa 적용)
- `/darkknight-worker` : 무거운 다운로드 및 AI 변환을 전담하는 백엔드 엔진 (Node.js Express)

## 🎯 현재 진행 상황 (완료된 작업)
1. **확장 프로그램 1클릭 파이프라인 연동 완료**
   - 확장 프로그램에서 비디오 주소(M3U8)와 쿠키를 추출하여 워커 엔진(`http://localhost:4000/process`)으로 `POST` 전송하도록 뚫어두었습니다.
2. **Next.js PWA SaaS 대시보드 스캐폴딩 완료**
   - 다크 모드, Glassmorphism이 적용된 대시보드 UI(`page.tsx`, `globals.css`) 구축 완료.
3. **Node.js 헤비 워커 엔진 (Worker) 세팅 완료**
   - Vercel 등 서버리스의 타임아웃 60초 한계를 우회하기 위해, 다운로드 및 FFmpeg 처리를 전담하는 Express 서버를 별도로 구축했습니다.

## 🛠 실행 방법
터미널 창을 2개 띄워서 각각 실행해야 합니다.
```bash
# 1. 프론트엔드 대시보드 실행 (포트 3000)
cd darkknight-saas
npm install
npm run dev --webpack

# 2. 백엔드 워커 엔진 실행 (포트 4000)
cd darkknight-worker
npm install
npm run dev
```

> [!WARNING]
> 이전 컴퓨터의 용량 문제로 `.gitignore`를 통해 `node_modules`와 `.next` 폴더가 모두 삭제된 채 푸시되었습니다. 새 컴퓨터에서 반드시 각 폴더별로 `npm install`을 다시 실행해 주십시오!
